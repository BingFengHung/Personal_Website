# Personal_Website
## 前言
我預計使用 Node.js 作為後端 Server，並使用 Vue 的框架來建立前端網頁

本網站主要是當作我履歷的網站，主要稍微介紹一下我所做過的工作與經歷，類似我的作品集~

期待之後可以增加更多

因為我不是很有錢，所以我預計將此網站建立在 Heroku 上面
基本上免費的話可以每個月可使用 550 MB 的流量 (應該有夠)

## 網站規劃
### 前端： Vue.js
預計顯使用最基本的 Vue.js 
之後有機會的話在使用 Vue CLI 建立完整的框架

### 後端： Node.js
使用 Express 製作 MVC 網站框架

### 資料庫： MongoDB
首次嘗試使用，我預計先在本地端將 MongoDB 的 Server 建立在 Docker 裡面，之後在加到 Heroku 上面

### 單元測試： Jest
我之前只有寫過 C# 的單元測試，透過這次的機會學習如何寫 JS 的單元測試

### 雲端伺服器
Heroku

## 開發環境
OS: Windows 10
Editor: Visual Code + Vim
Git: GitHub

## 網站開發時程
- 初期階段：我希望可以先以最快的速度 15 天將網站建立起來
- 中期階段：查看是否有其他可加入網站的元素 (Nginx)
- 長期階段：嘗試搭配 CI/CD 的方式 (沒接觸過)讓網站更穩定

基本上我會稍微寫一下開發過程中的相關資訊在 Note.md 檔案中，做一個開發紀錄
